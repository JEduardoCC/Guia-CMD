## COMANDOS CMD
```
mkdir "NOMBRE_CARPETA"
```
- Crea una carpeta
```
type nul > nombredearchivo.txt
```
- Crear un archivo
```
cd "NOMBRE_CARPETA"
```
- Moverse a la carpeta
```
cd ..
```
- Regresar a una carpeta anterior

```
start .
```
- Abrir la carpeta en el explorador
```
code .
```
- Abrir VSCode en la carpeta seleccionada
